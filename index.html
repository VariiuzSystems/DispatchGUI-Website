Branches

You can easily manage your Branches and can use all features which are included in the Dispatch.exe but only in a clear view. You have everything in one table and you are able to add multiple Application IDs which you can switch to see the other branches.

Builds

Mangage your Builds like you would to it in the Console but only easier. You can use all features and have an clear window where you can select what do to like "PUSH", "Mark as corrupt" and all other features. Also uses the Application ID you select.

Manifests

Get your Manifests in a easy to read List to make it easier to get what you need.

Config

Change your Config how you need them, add new Configs like templates to make creating and publishing faster and easier.
